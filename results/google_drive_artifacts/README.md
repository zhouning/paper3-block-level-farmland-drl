# Google Drive Provenance Artifacts

This folder stores small non-restricted artifacts recovered from the Google
Drive Paper3 result folder.

`colab_timing_500227108_legacy.json` records the timing metadata for an earlier
500227108 A100 run. The final manuscript uses the rebalanced
`results/blocks/township_500227108_v2` results, so this file is retained only as
provenance and should not be used to reproduce the final Table 4 values.
