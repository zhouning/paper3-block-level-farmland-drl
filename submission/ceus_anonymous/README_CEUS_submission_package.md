# CEUS Anonymous Review Package - Paper 3

This directory is the anonymous reviewer-facing Computers, Environment and Urban
Systems package for Paper 3:

`From Parcels to Blocks: Rescaling Deep Reinforcement Learning for Farmland Consolidation Planning through Spatial Abstraction`

It was created from the local Paper3 working source without modifying the
original source file. Author-identifying administrative files are intentionally
not included in this repository copy.

## Recommended Upload Mapping

| Submission system field | File or folder |
|---|---|
| Manuscript file, anonymous | `01_main_document_anonymous/manuscript_ceus_anonymous.pdf` |
| Editable manuscript source, if requested | `CEUS_paper3_latex_source_anonymous.zip` |
| Title page | `02_title_page/title_page_ceus_template.txt` (replace placeholders before upload) |
| Highlights | `03_highlights/highlights_ceus.txt` |
| Figure captions | `04_figure_captions/figure_captions_ceus.txt` |
| Figure files | `05_figures/*.png` |
| Data and code availability text | `07_declarations_and_checks/data_availability_ceus.txt` |
| Declaration of competing interest | `07_declarations_and_checks/competing_interest_ceus.txt` |
| Generative AI declaration | `07_declarations_and_checks/generative_ai_declaration_ceus.txt` |

## Directory Contents

- `01_main_document_anonymous/`: reviewer-facing anonymous main manuscript.
- `02_title_page/`: title-page template for the non-anonymous administrative upload.
- `03_highlights/`: Elsevier-style highlights.
- `04_figure_captions/`: figure captions supplied separately from figure files.
- `05_figures/`: individual figure files for upload.
- `07_declarations_and_checks/`: data/code availability text, declarations, and anonymous checklist.
- `CEUS_paper3_latex_source_anonymous.zip`: compressed editable source package for upload if requested.

## Important Notes

The anonymous manuscript and source have been retargeted from Land Use Policy to Computers, Environment and Urban Systems by changing the LaTeX `\journal{}` field and package documentation comments.

The data/code availability text no longer contains a placeholder. Before final
submission, insert the final anonymous repository URL only if the submission
system requires a live link. Administrative author files should be handled
outside this anonymous repository copy.

The title-page file included here is a template only. It must be populated with
actual author names, affiliations, addresses, and email information in the
submission system or in a separate non-anonymous administrative upload.
