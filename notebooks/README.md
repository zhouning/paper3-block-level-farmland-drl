# Sanitized Colab Run Notebooks

This folder contains sanitized provenance notebooks recovered from Google Drive:

- `paper3_105_sanitized.ipynb`
- `paper3_108_v2_sanitized.ipynb`

Execution outputs, execution counts, and Colab authorship metadata were removed.
The notebooks record the original Colab workflow used to launch A100 runs and
copy results back to Google Drive.

The notebooks refer to the original private `paper3_colab.zip`. That archive is
not included in the public repository because it contained restricted raw
geospatial data. For public reproducibility, use the repository scripts and
derived artifacts documented in `docs/COLAB_GUIDE.md` and `REPRODUCIBILITY.md`.
